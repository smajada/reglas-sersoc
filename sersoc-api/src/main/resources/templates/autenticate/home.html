Instrucciones

1. Crear sentencias SQL

Para crear consultas a la base de datos, dirígete al menú SQL > Crear Sentencia. Ahí podrás generar las consultas necesarias para la validación.

  1. Key o Nombre de la Sentencia: Escribe un identificador único para la sentencia.
  2. Script: Redacta la consulta en SQL en el apartado correspondiente.

2. Crear reglas

Para establecer una regla, sigue estos pasos:

  1. Nombre de la Regla: Escribe un nombre descriptivo.
  2. Tipos de Prestación: Selecciona los tipos de prestación a los que se aplicará la regla.
  3. Código JavaScript: Escribe el código JavaScript necesario para la ejecución de la regla.

3. Validar prestaciones

Cuando hagas clic en el botón "Validar" en la página de una prestación, serás redirigido a una nueva pantalla. En esta pantalla podrás ver el estado de las reglas:

  • Reglas no Validadas: Indicadas con el símbolo de error.
  • Reglas Validadas: Indicadas con el símbolo éxito.
  • Reglas con Advertencias: Indicadas con el símbolo correspondiente.
